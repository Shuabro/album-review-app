# Country Album Reviews Frontend

This is the Angular frontend for Country Album Reviews.

For the full project overview (roadmap, backend status, and goals), see the repository root README: [../README.md](../README.md).

## Prerequisites

- Node.js
- npm
- Angular CLI

## Install Dependencies

```bash
npm install
```

## Run Locally

```bash
npm start
```

Open http://localhost:4200 in your browser.

## Available Scripts

- `npm start` - Run the Angular dev server.
- `npm run build` - Build the frontend for production.
- `npm run watch` - Build in watch mode for development.
- `npm test` - Run unit tests (Karma/Jasmine).

## Frontend Stack

- Angular
- Angular Material
- TypeScript
- CSS
